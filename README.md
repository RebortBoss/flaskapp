**simpleBlog**


一个十分简单。易用的博客系统。使用flask 开发

>* 定时 抓取 github  [blog](https://github.com/xiyoulaoyuanjia/blog) 处的 文章。并存入数据库中

>* 采用  disqus 评论系统

>* 前端 加载日历与 微薄 显示

>* 显示部分阅读全文功能


**todu**

>* 首页分页功能
>* 标签墙功能
>* 添加本地日志功能
>* 按日期显示本地blog功能
>* 移植sae
